## Setup

  ##### Clone the repository
```bash
git clone https://github.com/ishandeveloper/BMI-Calculator-App.git
```
  ##### Move to the desired folder
```bash
cd \BMI-Calculator-App
```

  ##### To run the app, simply write
```bash
flutter run
```
